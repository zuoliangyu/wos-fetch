# Security Policy

## 支持范围

当前仓库暂无长期支持承诺，安全修复以最新主线版本为准。

## 报告方式

如果你发现以下问题，建议不要直接公开提交敏感细节：

- 凭据泄露
- 未授权访问
- 本地文件暴露
- 任意请求伪造
- 影响用户数据安全的问题

请通过私下渠道联系维护者，并尽量提供：

- 问题描述
- 影响范围
- 复现步骤
- 预期风险
- 相关日志或截图

## 处理原则

- 优先确认问题是否可稳定复现
- 优先处理真实的数据安全和权限边界问题
- 修复前不公开敏感利用细节

## 使用者责任

本项目用于本地辅助处理文献数据。

使用者应自行保护：

- API Key
- 本地浏览器登录态
- 导出的结果文件
- 包含全文内容的 ZIP 产物
