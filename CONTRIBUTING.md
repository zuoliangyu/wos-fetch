# Contributing

欢迎提交 Issue 和 Pull Request。

## 提交前建议

- 先阅读 `README.md`
- 先确认问题可以在当前版本稳定复现
- 修改范围尽量最小，避免顺手重构无关代码
- 涉及 WoS 自动化、浏览器控制、全文抓取时，说明复现步骤和运行环境

## Issue 建议包含

- 操作系统版本
- Python 版本
- 浏览器类型与版本
- 是否使用 WoS 调试端口
- 输入文件类型
- 复现步骤
- 实际结果
- 期望结果
- 相关报错截图或日志

## Pull Request 要求

- 变更应聚焦单一问题
- 保持现有代码风格和命名方式
- 补充必要的错误处理
- 不要提交敏感信息、账号、令牌、Cookies 或受限数据
- 不要提交 `.venv`、缓存文件、运行结果文件

## 合规说明

请不要提交任何会刻意绕过访问控制、验证码、人机校验或站点权限限制的改动。

与第三方站点相关的功能修改，应以合法访问、最小干预和可维护性为前提。
